专题一 传统发酵技术

果酒和果醋的制作

  • 菌种：
    • 果酒：酵母菌（真核生物，兼性厌氧），最适温度18~25℃。有氧时大量繁殖，无氧时发酵产生酒精。
    • 果醋：醋酸菌（原核生物，好氧菌），最适温度30~35℃。当氧气、糖源充足时，将葡萄糖分解成醋酸；当缺少糖源时，先将乙醇变为乙醛，再将乙醛变为醋酸。
  • 原理：
    • 葡萄酒：C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂（酵母菌无氧呼吸）。
    • 醋酸：C₂H₅OH + O₂ → CH₃COOH + H₂O（醋酸菌有氧呼吸）。
  • 材料：新鲜葡萄（或苹果），榨汁机，发酵瓶（可用带盖的玻璃瓶），醋酸菌（或暴露空气自然接种）。
  • 制作步骤（果酒）：
    1. 冲洗葡萄（去除枝梗，防止污染），榨汁装入发酵瓶（留1/3空间供CO₂排出）。
    2. 密闭发酵，温度控制在18~25℃。初期酵母菌有氧繁殖（可短时通气），后期无氧发酵产生酒精。
    3. 10~12天后，观察到气泡减少，液面不再翻腾，酒味浓郁，即可过滤装瓶。
  • 果醋制作：
    1. 在葡萄酒基础上，打开瓶盖，通入无菌空气，温度调至30~35℃。
    2. 加入醋酸菌液，或暴露于空气中（醋酸菌自然富集），每天搅拌1~2次增加溶氧。
    3. 7~8天后，醋酸生成，酸味明显，停止发酵。
  • 注意事项：
    • 发酵瓶要消毒（70%酒精擦拭或煮沸），防止杂菌污染。
    • 葡萄不能反复冲洗，避免洗掉野生酵母。
    • 果酒制作过程中，拧松瓶盖但不打开（排气），防止爆瓶。

腐乳的制作

  • 主要菌种：毛霉（属真菌），还包括青霉、酵母等。毛霉的最适温度15~18℃。
  • 原理：毛霉等产生的蛋白酶将豆腐中的蛋白质分解为小分子肽和氨基酸，脂肪酶将脂肪分解为甘油和脂肪酸，从而形成腐乳的特殊风味。
  • 材料：含水量70%左右的豆腐（过软不易成形），盐，卤汤（酒、香辛料等）。
  • 制作步骤：
    1. 豆腐块平放在笼屉内，保持温度15~18℃，自然接种或接种毛霉孢子。5天后豆腐表面长满白色菌丝。
    2. 加盐腌制：逐层加盐，盐量从下往上增加，腌制8天左右。盐能析出水分，抑制微生物生长，使腐乳成形。
    3. 配制卤汤：酒（12%左右）、香辛料（花椒、桂皮等）、调味品，将腌好的豆腐块放入卤汤中，密封。
    4. 后期发酵：室温下密封3~6个月，成熟。
  • 注意事项：
    • 豆腐水分过多易腐败，过少则口感硬。
    • 盐浓度过高影响风味，过低易变质。
    • 卤汤中酒可抑制微生物生长，同时赋予酒香。

泡菜的制作与亚硝酸盐检测

  • 菌种：乳酸菌（包括乳酸链球菌、乳酸杆菌等，厌氧或兼性厌氧），最适温度26~36℃。
  • 原理：无氧条件下乳酸菌将葡萄糖分解为乳酸。C₆H₁₂O₆ → 2C₃H₆O₃（乳酸）。
  • 材料：新鲜蔬菜（白菜、萝卜等），泡菜坛（带水封），盐水（5%~20%），香辛料。
  • 制作步骤：
    1. 蔬菜洗净切块，晾干；配制盐水（煮沸冷却后使用，杀灭杂菌）。
    2. 蔬菜装坛，压实，注入盐水淹没蔬菜，加香辛料，盖上坛盖，水封（保证无氧环境）。
    3. 室温发酵，7~10天即可食用。
  • 亚硝酸盐检测（格里斯试剂比色法）：
    • 原理：在盐酸酸化条件下，亚硝酸盐与对氨基苯磺酸发生重氮化反应，再与N-1萘基乙二胺盐酸盐结合形成玫瑰红色染料，与标准显色液比较，估算亚硝酸盐含量。
    • 步骤：泡菜研磨过滤→提取液→加入试剂显色→比色测定。
    • 变化规律：发酵初期亚硝酸盐含量上升（杂菌硝酸还原酶作用），中期下降（乳酸环境抑制杂菌，亚硝酸盐被分解），后期稳定低水平。

专题二 微生物培养与应用

微生物的实验室培养

  • 培养基：
    • 按物理性质：液体、半固体、固体（加琼脂）。
    • 按功能：基础、选择、鉴别培养基。
    • 成分：水、碳源、氮源、无机盐，有些需生长因子。
  • 无菌技术：
    • 消毒：用理化方法杀死部分微生物（不包括芽孢和孢子）。如煮沸、紫外线、酒精擦拭。
    • 灭菌：杀死所有微生物包括芽孢。方法：高压蒸汽灭菌（121℃ 15-30min）、干热灭菌（160-170℃ 2h）、灼烧灭菌（接种环）、过滤除菌（不耐热物质）。
  • 纯化方法：
    • 平板划线法：通过划线稀释菌种，最后得到单菌落。注意划线前后灼烧接种环，不能划破培养基。
    • 稀释涂布平板法：菌液梯度稀释，涂布到平板上，培养后计数菌落数。一般选择菌落数30~300的平板计数。
  • 菌种保存：
    • 短期：4℃斜面保存。
    • 长期：甘油管藏（-20℃或-80℃）。

土壤中分解尿素的细菌的分离与计数

  • 原理：选择培养基以尿素为唯一氮源，只有能合成脲酶的微生物才能生长。脲酶将尿素分解为氨，使pH升高，酚红指示剂变红。
  • 步骤：
    1. 土壤取样：从有动植物遗体处取土，铲去表层土。
    2. 制备土壤悬液：称取1g土壤，加入99mL无菌水，振荡，制成10⁻²稀释液，依次稀释至10⁻⁶。
    3. 涂布平板：取0.1mL稀释液涂布在以尿素为唯一氮源的培养基平板上。
    4. 培养计数：30~37℃培养1~2天，统计菌落数。计算公式：每克土壤中菌数 = (菌落数平均值 × 稀释倍数) / 涂布量(0.1mL换算为mL)。
  • 鉴别：在培养基中加入酚红，菌落周围出现红色环带，说明产生脲酶。

分解纤维素的微生物的分离

  • 原理：刚果红能与纤维素形成红色复合物，当纤维素被分解后，菌落周围出现透明圈。利用选择培养基（以纤维素为唯一碳源）富集。
  • 步骤：
    1. 土壤取样：富含纤维素的环境（如腐叶土）。
    2. 选择培养：将土样加入以纤维素为碳源的液体培养基中，摇床培养1~2天（富集纤维素分解菌）。
    3. 梯度稀释，涂布到刚果红纤维素培养基平板上。
    4. 挑选透明圈大的菌落（分解能力强）接种到斜面保存。
  • 刚果红染色法：
    • 先培养微生物，长出菌落后，加入刚果红溶液染色，1h后倒掉，再加入NaCl溶液脱色，出现透明圈。
    • 或直接在培养基中加入刚果红，但需注意染料与纤维素形成复合物可能影响菌落生长。

专题三 植物组织培养

菊花的组织培养

  • 原理：植物细胞具有全能性，离体的器官、组织或细胞在无菌条件下，通过激素诱导，能脱分化和再分化形成完整植株。
  • 培养基：MS培养基（含大量元素、微量元素、有机物），添加植物激素（生长素、细胞分裂素）。生长素/细胞分裂素比例高时利于生根，比例低时利于生芽。
  • 步骤：
    1. 外植体消毒：取菊花幼嫩茎段，流水冲洗，70%酒精消毒30s，无菌水冲洗，再用次氯酸钠溶液消毒2~3min，无菌水冲洗多次。
    2. 接种：将外植体切段（带节）接种到诱导培养基（MS+BA（细胞分裂素）+NAA（生长素））上。
    3. 培养：温度18~22℃，光照16h/d。先脱分化形成愈伤组织，再分化出丛芽。
    4. 生根培养：将丛芽切割转入生根培养基（1/2MS+NAA）诱导生根。
    5. 炼苗移栽：打开瓶盖适应环境，移栽到基质中。
  • 注意事项：
    • 激素浓度和比例要适宜，pH一般5.8。
    • 整个过程需无菌操作。

月季的花药培养

  • 原理：花药中的花粉（小孢子）是单倍体细胞，通过培养可发育成单倍体植株，再经秋水仙素处理获得纯合二倍体，缩短育种年限。
  • 材料选择：选择单核期（单核靠边期）的花蕾，此时花粉对离体刺激敏感。可通过醋酸洋红染色镜检确定发育时期。
  • 步骤：
    1. 花蕾消毒（同外植体消毒）。
    2. 无菌条件下剥离花药，接种到诱导培养基（MS + 激素）上。注意不要损伤花药，以免产生二倍体愈伤组织。
    3. 培养：温度25℃左右，初期暗培养利于愈伤组织形成，后见光分化。
    4. 花粉胚或愈伤组织分化出单倍体植株。
    5. 染色体加倍：用秋水仙素处理幼苗，获得纯合二倍体。
  • 影响因素：亲本基因型、花粉发育时期、培养基组成、培养条件等。

专题四 酶的研究与应用

果胶酶在果汁生产中的作用

  • 果胶：植物细胞壁和胞间层成分，不溶于水，使果汁浑浊。果胶酶包括多聚半乳糖醛酸酶、果胶分解酶、果胶酯酶等。
  • 作用：分解果胶，使果汁澄清，提高出汁率。
  • 实验探究：探究温度、pH对果胶酶活性的影响；探究酶用量。用榨取的苹果泥加酶处理后，测量滤液体积或观察澄清度。
  • 注意事项：果胶酶最适温度40~50℃，最适pH 3.5~5.5。酶用量需通过实验确定最适范围。

探讨加酶洗衣粉的洗涤效果

  • 酶的种类：蛋白酶（分解蛋白质污渍）、脂肪酶（分解油渍）、淀粉酶（分解淀粉类污渍）、纤维素酶（使纤维蓬松，易去污）。
  • 实验设计：用不同加酶洗衣粉洗涤相同污布，控制水温、时间、水量等变量，比较清洁效果。注意设置对照（普通洗衣粉）。
  • 注意事项：
    • 酶在高温下失活，水温不宜过高（一般40℃左右）。
    • 蛋白酶不能洗涤真丝、羊毛织物（蛋白质纤维会被分解）。
    • 酶的活性受pH影响，洗衣粉多呈碱性。

酵母细胞的固定化

  • 固定化方法：包埋法（常用海藻酸钠）、化学结合法、物理吸附法。细胞多采用包埋法，因为体积大，酶多用结合法。
  • 固定化酵母细胞步骤：
    1. 酵母活化：干酵母加入蒸馏水，搅拌，1h后备用。
    2. 配制海藻酸钠溶液：0.7g海藻酸钠溶于10mL水，小火加热溶解，冷却至室温。
    3. 混合：将活化酵母细胞加入海藻酸钠溶液中，混匀。
    4. 固定：用注射器将混合液缓慢滴入CaCl₂溶液中，形成凝胶珠（钙离子与海藻酸根交联）。
    5. 洗涤备用：用蒸馏水冲洗凝胶珠，去除CaCl₂。
  • 应用：固定化酵母可用于连续发酵生产酒精、啤酒等，易于产物分离和酶回收。
  • 注意事项：
    • 海藻酸钠浓度不宜过高（不易成形）或过低（珠不牢固）。
    • 混合液温度不能太高，以免杀死酵母。
    • CaCl₂浓度一般0.05mol/L。

专题五 DNA和蛋白质技术

DNA的粗提取与鉴定

  • 材料：鸡血（哺乳动物红细胞无细胞核，不宜用）、菜花、香蕉等。
  • 原理：
    • DNA在2mol/L NaCl溶液中溶解度大，在0.14mol/L NaCl中溶解度最低（析出）。
    • DNA不溶于冷酒精（蛋白质等可溶），可进一步纯化。
    • 鉴定：二苯胺试剂（沸水浴）使DNA变蓝色。
  • 步骤（以鸡血为例）：
    1. 制备鸡血细胞液：加柠檬酸钠抗凝，离心取下层细胞。
    2. 释放DNA：加蒸馏水使细胞吸水涨破，搅拌后用纱布过滤。
    3. 溶解核内DNA：加入2mol/L NaCl溶液，使DNA溶解。
    4. 析出DNA：加蒸馏水稀释至0.14mol/L NaCl，DNA析出（用玻璃棒卷起）。
    5. 再溶解：将丝状物溶于2mol/L NaCl。
    6. 纯化：加冷酒精（95%），DNA沉淀，用玻璃棒挑出。
    7. 鉴定：取少量DNA溶于NaCl，加入二苯胺试剂，沸水浴5min，观察蓝色。

多聚酶链式反应扩增DNA片段

  • 原理：DNA体外复制，利用DNA聚合酶（Taq酶，耐高温），以少量DNA为模板，在引物引导下，通过高温变性、低温复性、中温延伸三步循环，大量扩增目的DNA。
  • 反应体系：模板DNA、引物（两种）、Taq酶、脱氧核苷酸（dNTP）、缓冲液（含Mg²⁺）。
  • 基本步骤：
    • 变性：90~95℃，DNA双链解开。
    • 复性：55~60℃，引物与模板链结合。
    • 延伸：70~75℃，Taq酶催化合成互补链。
    • 循环30次左右，DNA扩增约2ⁿ倍。
  • 结果检测：琼脂糖凝胶电泳，用溴化乙锭（EB）染色，紫外灯下观察DNA条带。
  • 注意事项：
    • 引物设计要避免自身互补，长度18-25bp。
    • 防止外源DNA污染，使用无菌器材。

血红蛋白的提取和分离

  • 材料：新鲜猪血（加抗凝剂）。
  • 步骤：
    1. 样品处理：离心分离红细胞，洗涤红细胞（加生理盐水），加蒸馏水（低渗）和甲苯（使细胞破裂），释放血红蛋白。
    2. 粗分离：离心去除细胞碎片，得到血红蛋白溶液。
    3. 透析：将溶液装入透析袋，置于磷酸缓冲液中，去除小分子杂质。
    4. 凝胶色谱法纯化：根据分子大小分离蛋白质。大分子先流出，小分子后流出。常用凝胶如Sephadex G-75。
    5. 纯度鉴定：SDS-聚丙烯酰胺凝胶电泳（SDS-PAGE），根据分子量分离蛋白质条带。
  • 注意事项：
    • 整个过程在低温下进行（防止蛋白质变性）。
    • 凝胶色谱柱装填要均匀，不能有气泡。
    • 洗脱液流速要稳定。

专题六 植物有效成分的提取

植物芳香油的提取

  • 方法：
    • 水蒸气蒸馏法：利用水蒸气将挥发性较强的芳香油携带出来，冷凝后分离。适用于挥发性高的精油（如玫瑰油、薄荷油）。分为水中蒸馏、水上蒸馏、水气蒸馏。
    • 压榨法：通过机械压榨，将柑橘、柠檬等果皮中的芳香油压出，分离。
    • 有机溶剂萃取法：用石油醚、乙醇等有机溶剂浸泡原料，蒸去溶剂得到芳香油。适用于挥发性较低、热敏感的成分。
  • 实验操作（以玫瑰精油为例）：
    1. 鲜玫瑰花放入蒸馏瓶，加水，加热蒸馏。
    2. 冷凝管收集馏出液，得到乳白色的油水混合物。
    3. 用分液漏斗分离出油层，加入无水硫酸钠吸水，得到精油。
  • 注意事项：
    • 蒸馏时间不宜过长，防止有效成分分解。
    • 压榨法需用石灰水浸泡，破坏细胞结构，提高出油率。

胡萝卜素的提取

  • 性质：胡萝卜素是脂溶性物质，化学性质稳定，不溶于水，微溶于乙醇，易溶于石油醚等有机溶剂。β-胡萝卜素在人体内可转化为维生素A。
  • 提取方法：常用有机溶剂萃取法，因为胡萝卜素挥发性差，不能用水蒸气蒸馏。
  • 步骤：
    1. 材料处理：胡萝卜粉碎，干燥（脱水，利于萃取）。
    2. 萃取：将胡萝卜粉装入萃取装置，加入石油醚，水浴加热回流萃取。
    3. 过滤：除去固体残渣，得到萃取液。
    4. 浓缩：用蒸馏装置蒸发石油醚，得到胡萝卜素粗品。
  • 鉴定：纸层析法，以标准胡萝卜素样品对照，观察层析带的位置和颜色。
  • 注意事项：
    • 萃取剂应选择沸点低、毒性小、不易燃、对胡萝卜素溶解度大的（如石油醚）。
    • 避免明火加热，使用水浴防止着火。
    • 干燥温度不宜过高，防止胡萝卜素分解。
