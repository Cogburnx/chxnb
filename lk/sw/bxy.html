第一章 走进细胞

细胞学说及其建立

  • 建立者：施莱登、施旺（19世纪）。
  • 细胞学说要点：
    • 细胞是一个有机体，一切动植物都由细胞发育而来，并由细胞和细胞产物所构成。
    • 细胞是一个相对独立的单位，既有它自己的生命，又对与其他细胞共同组成的整体生命起作用。
    • 新细胞是由老细胞分裂产生的（魏尔肖总结）。
  • 意义：
    • 揭示了动物和植物的统一性，从而阐明了生物界的统一性。
    • 为进化论的确立奠定了基础，标志着生物学研究进入细胞水平。

原核细胞与真核细胞

  • 主要区别：有无以核膜为界限的细胞核。
  • 原核细胞：
    • 无核膜、核仁，只有一个环状DNA分子位于拟核区域。
    • 细胞器只有核糖体（70S），无膜包细胞器。
    • 细胞壁主要成分是肽聚糖（细菌）。
    • 代表生物：细菌、蓝细菌（蓝藻）、支原体（无细胞壁）、衣原体、放线菌。
    • 蓝细菌含有藻蓝素和叶绿素，能进行光合作用，但无叶绿体。
  • 真核细胞：
    • 有核膜包被的细胞核，有核仁，DNA与蛋白质结合形成染色体。
    • 有多种细胞器（线粒体、叶绿体、内质网、高尔基体等）。
    • 植物细胞壁主要成分是纤维素和果胶；真菌细胞壁主要成分为几丁质。
    • 代表生物：动物、植物、真菌（酵母、蘑菇）、原生生物。

细胞的多样性与统一性

  • 多样性：形态、大小、结构、功能差异巨大（如红细胞、神经细胞、肌肉细胞）。
  • 统一性：
    • 都有相似的细胞膜、细胞质（含核糖体），都以DNA作为遗传物质。
    • 都进行细胞呼吸（能量代谢），共用一套遗传密码。

显微镜的使用

  • 放大倍数 = 目镜放大倍数 × 物镜放大倍数。
  • 成像特点：倒立放大的虚像，标本移动方向与物像移动方向相反。
  • 使用步骤：
    1. 低倍镜观察：对光→放置玻片→调节粗准焦螺旋找到物像→调节细准焦螺旋至清晰。
    2. 高倍镜观察：将目标移至视野中央→转动转换器换高倍镜→调节细准焦螺旋（不能动粗准焦）→调节光圈和反光镜使视野明亮。
  • 注意事项：
    • 观察颜色浅的标本时视野应调暗（平面反光镜、小光圈）。
    • 高倍镜下只能使用细准焦螺旋调焦。

第二章 组成细胞的分子

元素和化合物概述

  • 大量元素（含量占生物体总量万分之一以上）：C、H、O、N、P、S、K、Ca、Mg等。
  • 微量元素：Fe、Mn、B、Zn、Mo、Cu（铁猛碰新木桶）。
  • 最基本元素：C（因为碳原子易形成四个化学键，构成有机物的骨架）。
  • 化合物：无机物（水、无机盐）、有机物（蛋白质、脂质、糖类、核酸）。含量最多的是水，占85%~90%；有机物中最多的是蛋白质。

无机物：水、无机盐

  • 水：
    • 结合水：与细胞内其他物质结合，是细胞结构的重要组成成分（约4.5%）。
    • 自由水：良好溶剂，参与化学反应，运输营养物质和废物，维持细胞形态。自由水/结合水比值越高，代谢越旺盛。
  • 无机盐：
    • 存在形式：多数以离子形式存在（Na⁺、K⁺、Ca²⁺、Mg²⁺、Cl⁻、HCO₃⁻、HPO₄²⁻）。
    • 功能：
      • 组成某些重要化合物（如Mg²⁺是叶绿素的必需成分；Fe²⁺是血红蛋白的组分；I⁻是甲状腺激素的组分）。
      • 维持细胞和生物体的生命活动（如哺乳动物血钙过低会抽搐）。
      • 维持渗透压和酸碱平衡。

糖类与脂质

  • 糖类（C、H、O）：
    • 单糖：葡萄糖（细胞主要能源物质）、果糖、半乳糖、核糖、脱氧核糖。
    • 二糖：蔗糖（葡萄糖+果糖）、麦芽糖（葡萄糖+葡萄糖）、乳糖（葡萄糖+半乳糖）。
    • 多糖：淀粉（植物储能）、糖原（动物储能，包括肝糖原、肌糖原）、纤维素（植物细胞壁主要成分，不供能）、几丁质（甲壳类外壳、真菌细胞壁）。
  • 脂质（C、H、O，有的含N、P）：
    • 脂肪：储能，保温，缓冲减压，由甘油和脂肪酸组成。
    • 磷脂：构成生物膜（细胞膜、细胞器膜）的重要成分，含磷酸和甘油。
    • 固醇：包括胆固醇（构成动物细胞膜，参与血液中脂质运输）、性激素（促进生殖器官发育和生殖细胞形成）、维生素D（促进肠道对钙磷吸收）。

蛋白质（结构、功能、检测）

  • 基本单位：氨基酸（约20种）。结构通式：中心碳原子连接氨基（-NH₂）、羧基（-COOH）、氢原子（H）和侧链基团（R基）。不同的氨基酸R基不同。
  • 脱水缩合：一个氨基酸的羧基与另一个氨基酸的氨基脱去一分子水，形成肽键（-CO-NH-）。
  • 蛋白质结构层次：
    • 一级结构：氨基酸的排列顺序（肽链）。
    • 二级结构：肽链盘曲、折叠形成α-螺旋或β-折叠（由氢键维系）。
    • 三级结构：肽链进一步盘曲形成特定空间结构（由二硫键、氢键、疏水作用等维系）。
    • 四级结构：多条肽链聚合形成特定构象（如血红蛋白）。
  • 结构多样性原因：氨基酸种类、数目、排列顺序不同，肽链的盘曲、折叠方式及其空间结构不同。
  • 功能：
    • 结构蛋白：构成细胞和生物体结构（如羽毛、肌肉）。
    • 催化：绝大多数酶是蛋白质。
    • 运输：血红蛋白、载体蛋白。
    • 调节：胰岛素、生长激素等蛋白质类激素。
    • 免疫：抗体（免疫球蛋白）。
  • 检测：双缩脲试剂（A液：0.1g/mL NaOH；B液：0.01g/mL CuSO₄）。先加A液营造碱性环境，再加B液，出现紫色反应（与肽键结合）。

核酸（DNA、RNA）

  • 基本单位：核苷酸，由磷酸、五碳糖、含氮碱基组成。
  • DNA（脱氧核糖核酸）：
    • 五碳糖为脱氧核糖，碱基为A、T、C、G（无U）。
    • 通常为双螺旋结构，主要分布在细胞核（线粒体、叶绿体也有）。
    • 功能：携带遗传信息，决定遗传性状。
  • RNA（核糖核酸）：
    • 五碳糖为核糖，碱基为A、U、C、G（无T）。
    • 通常为单链，主要分布在细胞质。
    • 种类：mRNA（信使RNA，携带DNA遗传信息）、tRNA（转运RNA，运输氨基酸）、rRNA（核糖体RNA，组成核糖体）。

有机物检测实验

  • 还原糖检测（斐林试剂）：
    • 斐林试剂甲液（0.1g/mL NaOH）、乙液（0.05g/mL CuSO₄），等量混合后使用（现配现用）。
    • 与还原糖（葡萄糖、果糖、麦芽糖）在水浴加热条件下生成砖红色沉淀（Cu₂O）。
    • 甘蔗茎的薄壁组织、甜菜块根含蔗糖（非还原糖），不宜做材料。
  • 脂肪检测（苏丹Ⅲ或苏丹Ⅳ染液）：
    • 苏丹Ⅲ染液将脂肪染成橘黄色，苏丹Ⅳ染成红色。
    • 可用花生种子切片（需用50%酒精洗去浮色）或组织样液（直接观察）。
  • 蛋白质检测（双缩脲试剂）：
    • 先加A液（NaOH）摇匀，再加B液（CuSO₄）摇匀，出现紫色。
  • 淀粉检测：碘液，变蓝。

第三章 细胞的基本结构

细胞膜（流动镶嵌模型）

  • 成分：脂质（约50%，磷脂为主）、蛋白质（约40%）、糖类（约2%~10%，与蛋白质或脂质结合形成糖蛋白、糖脂）。
  • 流动镶嵌模型要点：
    • 磷脂双分子层构成基本骨架，具有流动性（头部亲水向外，尾部疏水向内）。
    • 蛋白质分子镶嵌、贯穿或覆盖在磷脂双分子层表面，大多数蛋白质可以运动。
    • 细胞膜外表有糖被（糖蛋白），与细胞识别、信息交流有关。
  • 功能：
    • 将细胞与外界环境分隔开，保障细胞内部环境的相对稳定。
    • 控制物质进出细胞（选择透过性）。
    • 进行细胞间的信息交流（如受体识别、胞间连丝）。

细胞器（结构与功能）

  • 双层膜细胞器：
    • 线粒体：有氧呼吸主要场所，含DNA、RNA和核糖体（半自主细胞器）。内膜向内折叠形成嵴，增大膜面积。细胞所需能量95%来自线粒体。
    • 叶绿体（植物特有）：光合作用场所，含DNA、RNA、核糖体，类囊体堆叠成基粒，增大了膜面积。
  • 单层膜细胞器：
    • 内质网：粗面内质网（附着核糖体，参与蛋白质合成与加工），滑面内质网（合成脂质、解毒）。
    • 高尔基体：对蛋白质进行加工、分类、包装；与植物细胞壁形成有关（分泌纤维素）。
    • 溶酶体：含多种水解酶，分解衰老损伤细胞器，吞噬并杀死侵入细胞的病毒/细菌。
    • 液泡（植物细胞）：内含细胞液（糖类、无机盐、色素、蛋白质），调节渗透压，维持细胞形态。
    • 核糖体（无膜）：由rRNA和蛋白质组成，蛋白质合成场所，分为附着核糖体和游离核糖体。
    • 中心体（动物、低等植物）：由两个互相垂直的中心粒组成，与有丝分裂有关（发出星射线形成纺锤体）。
  • 细胞骨架：由蛋白质纤维组成的网架结构，维持细胞形态，与运动、分裂、分化、物质运输等有关。

细胞核（核孔、染色质）

  • 结构：
    • 核膜：双层膜，把核内物质与细胞质分开。上有核孔（大分子物质通道，如mRNA出核，蛋白质进核）。
    • 核仁：与某种RNA（rRNA）的合成及核糖体形成有关。
    • 染色质：由DNA和蛋白质组成，易被碱性染料染色。细胞分裂时高度螺旋化为染色体（同一物质在不同时期的两种形态）。
  • 功能：遗传信息库，是细胞代谢和遗传的控制中心。

生物膜系统

  • 组成：细胞膜、核膜、细胞器膜（线粒体、叶绿体、内质网、高尔基体、溶酶体、液泡等膜）共同构成。
  • 功能：
    • 使细胞具有相对稳定的内部环境，同时分隔各种细胞器，保证生命活动高效有序进行。
    • 为多种酶提供附着位点，利于化学反应进行。
    • 参与细胞间的信息交流、物质运输、能量转换等。

第四章 物质的输入和输出

被动运输（自由扩散、协助扩散）

  • 自由扩散：
    • 物质顺浓度梯度运输，不消耗能量，不需要转运蛋白。
    • 实例：O₂、CO₂、甘油、乙醇、苯等脂溶性小分子。
  • 协助扩散：
    • 顺浓度梯度，不消耗能量，需要膜上的转运蛋白（载体蛋白或通道蛋白）。
    • 实例：水通道蛋白运输水（有些水分子也可自由扩散）；葡萄糖进入红细胞（载体蛋白）；离子通道（Na⁺、K⁺通道）。

主动运输与胞吞胞吐

  • 主动运输：
    • 逆浓度梯度，需要载体蛋白，消耗能量（ATP）。
    • 实例：Na⁺、K⁺等离子通过钠钾泵（消耗ATP，排出3个Na⁺，泵入2个K⁺）；葡萄糖、氨基酸进入小肠上皮细胞。
  • 胞吞、胞吐：
    • 大分子或颗粒物质，依赖细胞膜的流动性，消耗能量，需要膜上蛋白质参与（但不属于跨膜运输）。
    • 胞吞：细胞外物质形成小囊泡进入细胞（如吞噬细胞吞噬细菌）。
    • 胞吐：细胞内囊泡与细胞膜融合，将物质排出（如分泌蛋白、神经递质）。

渗透作用与质壁分离

  • 渗透作用：溶剂分子（水）通过半透膜的扩散。条件：半透膜、膜两侧浓度差。
  • 动物细胞：红细胞在清水中吸水涨破，在浓盐水中失水皱缩。
  • 植物细胞质壁分离：
    • 原理：成熟植物细胞的原生质层（细胞膜、液泡膜及之间的细胞质）相当于半透膜，细胞液与外界溶液存在浓度差。
    • 外界溶液浓度 > 细胞液浓度 → 细胞失水，发生质壁分离（原生质层与细胞壁分离）。
    • 质壁分离复原：将分离的细胞放入清水或低渗溶液，细胞吸水恢复原状。
    • 应用：判断细胞死活（只有活细胞才能发生质壁分离）；测定细胞液浓度范围。

第五章 细胞的能量供应和利用

酶（特性、影响因素、实验）

  • 本质：绝大多数是蛋白质，少数是RNA（核酶）。
  • 作用机理：显著降低化学反应的活化能（与无机催化剂相比，酶降低活化能的效果更显著）。
  • 特性：
    • 高效性：比无机催化剂效率高10⁷~10¹³倍。
    • 专一性：一种酶只能催化一种或一类底物。
    • 作用条件温和：高温、强酸、强碱会使酶变性失活；低温使酶活性降低但不变性。
  • 影响酶活性的因素：
    • 温度：最适温度下活性最高，低温抑制，高温失活。
    • pH：最适pH下活性最高，过酸过碱失活（不同酶最适pH不同，如胃蛋白酶最适pH 1.5~2.0，唾液淀粉酶最适pH 6.8）。
    • 酶浓度、底物浓度：在底物充足时酶促反应速率与酶浓度成正比；酶量一定时，底物浓度低时速率与底物浓度成正比，底物浓度过高则速率趋于最大（饱和）。
  • 实验设计（探究温度对酶活性影响）：
    • 常用淀粉酶和淀粉，避免用H₂O₂（H₂O₂加热会自行分解）。
    • 检测试剂用碘液（不能用斐林试剂，因斐林试剂需要加热干扰温度变量）。

ATP（结构、转化、利用）

  • 结构：腺苷三磷酸（A—P~P~P）。A代表腺苷（腺嘌呤+核糖），P代表磷酸基团，~代表特殊的化学键（高能磷酸键）。
  • ATP与ADP的相互转化：
    • ATP + H2O → ADP + Pi + 能量。水解时远离A的高能磷酸键断裂，释放能量。
    • 合成ATP的能量来源：动物通过呼吸作用（细胞呼吸）；植物通过呼吸作用和光合作用。
  • ATP的利用：直接能源物质。用于主动运输、肌肉收缩、发光、发电、物质合成、神经传导、细胞分裂等几乎所有生命活动。

细胞呼吸（有氧、无氧）

  • 有氧呼吸：
    • 场所：细胞质基质（第一阶段）、线粒体基质（第二阶段）、线粒体内膜（第三阶段）。
    • 第一阶段：C₆H₁₂O₆ → 2丙酮酸 + 4[H] + 少量能量（2ATP）。
    • 第二阶段：2丙酮酸 + 6H₂O → 6CO₂ + 20[H] + 少量能量（2ATP）。
    • 第三阶段：24[H] + 6O₂ → 12H₂O + 大量能量（34ATP）。
    • 总反应式：C₆H₁₂O₆ + 6H₂O + 6O₂ → 6CO₂ + 12H₂O + 能量（约38ATP，实际约30~32）。
  • 无氧呼吸：
    • 场所：细胞质基质。
    • 第一阶段与有氧呼吸相同：C₆H₁₂O₆ → 2丙酮酸 + 4[H] + 2ATP。
    • 第二阶段丙酮酸在不同酶作用下产生不同产物：
      • 乳酸发酵：2丙酮酸 + 4[H] → 2乳酸（如动物、马铃薯块茎、乳酸菌）。
      • 酒精发酵：2丙酮酸 → 2C₂H₅OH + 2CO₂（如植物、酵母菌）。
    • 总反应式（乳酸）：C₆H₁₂O₆ → 2C₃H₆O₃ + 少量能量（2ATP）。
    • 总反应式（酒精）：C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 少量能量（2ATP）。
  • 应用：
    • 中耕松土（增加土壤氧气，促进根系有氧呼吸）。
    • 低温低氧储存种子、水果（抑制呼吸作用，减少有机物消耗）。
    • 伤口用透气纱布（抑制厌氧菌繁殖）。
    • 酿酒、制醋、泡菜等发酵工艺。
  • 探究实验（酵母菌呼吸方式）：
    • 产物检测：澄清石灰水变浑浊（CO₂）；酸性重铬酸钾（橙色→灰绿色）用于检测酒精（需在酸性条件下）。

光合作用（色素、过程、因素）

  • 色素（叶绿体中）：
    • 叶绿素a（蓝绿色）、叶绿素b（黄绿色）——主要吸收蓝紫光和红光。
    • 类胡萝卜素（胡萝卜素橙黄色、叶黄素黄色）——主要吸收蓝紫光。
    • 色素分布于类囊体薄膜上，可用纸层析法分离（溶解度高的在滤纸条上方）。
  • 光反应（类囊体薄膜）：
    • 条件：光、色素、酶、水。
    • 过程：水的光解（产生O₂、[H]即NADPH）；合成ATP（光能→电能→活跃化学能）。
    • 产物：O₂、[H]（NADPH）、ATP（均供给暗反应）。
  • 暗反应（叶绿体基质）：
    • 条件：酶、CO₂、[H]、ATP。
    • 过程：CO₂的固定（CO₂ + C₅ → 2C₃）；C₃的还原（2C₃ + [H] + ATP → (CH₂O) + C₅）。
    • 产物：有机物（如葡萄糖）、ADP、Pi（回到光反应）。
  • 总反应式：CO₂ + H₂O →(光照、酶、叶绿体) (CH₂O) + O₂。
  • 影响因素：
    • 光照强度：一定范围内光合速率随光照增强而加快，超过光饱和点不再增加。
    • CO₂浓度：一定范围内随CO₂浓度增加而加快，达到CO₂饱和点后不变。
    • 温度：影响酶活性，有最适温度。
    • 水、矿质元素等。
  • 应用：温室大棚提高CO₂浓度（通风、施有机肥）、适当增加光照强度、控制温度等。

第六章 细胞的生命历程

细胞的增殖（有丝分裂）

  • 细胞周期：连续分裂的细胞，从一次分裂完成时开始，到下一次分裂完成时为止。包括分裂间期（G₁、S、G₂）和分裂期（M期）。
  • 分裂间期：G₁期（合成蛋白质，为DNA复制做准备）、S期（DNA复制）、G₂期（合成蛋白质，为分裂期做准备）。
  • 有丝分裂过程（以植物细胞为例）：
    • 前期：染色质→染色体（螺旋化），核仁消失，核膜解体，出现纺锤体（两极发出纺锤丝）。
    • 中期：染色体着丝粒排列在赤道板上，染色体形态稳定、数目清晰，便于观察。
    • 后期：着丝粒分裂，姐妹染色单体分开成为子染色体，在纺锤丝牵引下移向两极。
    • 末期：染色体→染色质，核膜重建，核仁出现，纺锤体消失，赤道板位置出现细胞板（高尔基体参与），形成新细胞壁。
  • 动物细胞与植物细胞有丝分裂区别：
    • 前期：动物细胞中心体发出星射线形成纺锤体；植物细胞由两极发出纺锤丝。
    • 末期：动物细胞不形成细胞板，细胞膜向内凹陷缢裂成两个细胞。
  • 意义：将亲代染色体复制后精确平均分配到两个子细胞，保持了遗传稳定性。

细胞分化与全能性

  • 细胞分化：在个体发育中，由一个或一种细胞增殖产生的后代，在形态、结构和生理功能上发生稳定性差异的过程。
  • 实质：基因的选择性表达（奢侈基因表达，管家基因都表达），遗传物质不变。
  • 意义：形成不同组织器官，是个体发育的基础。
  • 细胞全能性：已经分化的细胞，仍然具有发育成完整个体的潜能。
  • 实例：
    • 植物细胞全能性易体现（如组织培养）。
    • 动物细胞核具有全能性（克隆羊多莉）。
    • 胚胎干细胞（全能或高全能性），成体干细胞（多能或专能）。

细胞的衰老与死亡

  • 衰老细胞特征：
    • 水分减少，体积变小，代谢减慢。
    • 多种酶活性降低（如酪氨酸酶活性降低→白发）。
    • 色素积累（如老年斑）。
    • 细胞核体积增大，核膜内折，染色质收缩。
    • 细胞膜通透性改变，物质运输功能降低。
  • 细胞凋亡：
    • 由基因决定的细胞自动结束生命的过程（程序性死亡）。
    • 对生物体有利，如蝌蚪尾巴消失、胎儿指间细胞凋亡、被病原体感染的细胞清除。
  • 细胞坏死：在不利因素下（如缺氧、高温、机械损伤）引起的细胞被动死亡，细胞膜破裂，内容物外泄，引发炎症。

癌细胞（特征、成因）

  • 主要特征：
    • 无限增殖（不受机体控制）。
    • 形态结构改变（如球形）。
    • 细胞膜上糖蛋白减少，黏着性降低，易分散转移。
  • 致癌因子：
    • 物理致癌因子（紫外线、X射线）。
    • 化学致癌因子（亚硝胺、黄曲霉毒素）。
    • 病毒致癌因子（Rous肉瘤病毒）。
  • 原癌基因与抑癌基因：
    • 原癌基因：调控细胞正常生长和分裂的基因，突变后导致过度增殖。
    • 抑癌基因：抑制细胞增殖，促进细胞分化和凋亡的基因，突变后失去抑制作用。
    • 癌症的发生是多个基因突变累积的结果。
