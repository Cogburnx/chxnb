第一章 遗传因子的发现

分离定律

  • 孟德尔一对相对性状杂交实验
    • 选材：豌豆（严格自花传粉、闭花授粉；具有易于区分的相对性状；生长周期短，子代数量多）。
    • 过程：纯种高茎（DD） × 纯种矮茎（dd） → F₁全为高茎（Dd） → F₁自交 → F₂出现性状分离，高茎:矮茎 ≈ 3:1。
    • 解释：遗传因子成对存在；形成配子时成对因子分离；受精时雌雄配子随机结合。
  • 分离定律实质：在杂合子细胞中，位于一对同源染色体上的等位基因具有一定的独立性；在减数分裂形成配子时，等位基因随同源染色体的分开而分离，分别进入两个配子中，独立地随配子遗传给后代。
  • 验证方法：测交（F₁与隐性纯合子交配），后代比例应为1:1，证明F₁产生两种配子。
  • 显性的相对性：
    • 完全显性：F₁表现出与显性亲本完全相同的性状（如豌豆株高）。
    • 不完全显性：F₁表现为双亲中间性状（如紫茉莉花色，红花×白花→粉红花）。
    • 共显性：双亲性状同时在F₁个体上表现出来（如ABO血型中的IAIB）。

自由组合定律

  • 两对相对性状杂交实验
    • 纯种黄色圆粒（YYRR）× 绿色皱粒（yyrr） → F₁全为黄色圆粒（YyRr）。
    • F₁自交 → F₂出现四种表现型：黄圆:黄皱:绿圆:绿皱 ≈ 9:3:3:1。
  • 解释：
    • F₁产生配子时，每对等位基因分离，不同对基因自由组合，产生YR、Yr、yR、yr四种比例相等的配子。
    • 受精时，16种雌雄配子随机结合，组合方式共16种，基因型9种，表现型4种。
  • 自由组合定律实质：控制不同性状的基因（非等位基因）在减数分裂过程中，非同源染色体上的非等位基因自由组合。
  • 验证：测交（F₁与双隐性个体杂交），后代四种表现型比例应为1:1:1:1。
  • 应用：指导杂交育种（将不同亲本的优良性状组合到一起）；遗传病预测（两种遗传病同时出现的概率计算）。

显性的相对性

  • 完全显性：具有相对性状的两个亲本杂交，F₁与显性亲本表现一致。如豌豆的高茎对矮茎。
  • 不完全显性：F₁表现为双亲的中间类型。如金鱼草花色（红花×白花→粉红花）。此时F₂分离比为1:2:1（表现型与基因型一致）。
  • 共显性：两个亲本的性状同时在F₁个体上显现。如人类的MN血型（LMLN基因型表现为MN血型，同时有M和N抗原）。
  • 镶嵌显性：双亲性状在F₁不同部位镶嵌表现（如异色瓢虫鞘翅色斑）。

第二章 基因和染色体的关系

减数分裂与受精作用

  • 减数分裂概念：进行有性生殖的生物，在产生成熟生殖细胞时，染色体数目减半的分裂。
  • 过程（以精子形成过程为例）：
    • 减数第一次分裂：
      • 间期：DNA复制，有关蛋白质合成，染色体数目不变（DNA加倍）。
      • 前期I：联会（同源染色体两两配对），形成四分体（每个四分体含4条染色单体，由一对同源染色体组成），可能发生交叉互换。
      • 中期I：同源染色体成对排列在赤道板两侧。
      • 后期I：同源染色体分离，非同源染色体自由组合，分别移向两极。
      • 末期I：形成两个子细胞，染色体数目减半。
    • 减数第二次分裂：类似有丝分裂，着丝粒分裂，姐妹染色单体分开，形成四个精细胞，变形后成为精子。
  • 卵细胞形成过程：与精子形成相似，但细胞质不均等分裂，产生一个卵细胞和三个极体。
  • 受精作用：精子和卵细胞融合，恢复体细胞染色体数目（2n），同时体现了有性生殖的多样性（配子多样性 + 受精随机性）。
  • 意义：减数分裂和受精作用保证了生物前后代染色体数目的恒定，为生物变异提供了可能。

基因在染色体上

  • 萨顿假说：基因和染色体行为存在明显的平行关系（基因在杂交过程中保持完整和独立性，染色体在配子形成和受精过程中也有相对稳定的形态结构；体细胞中基因成对存在，染色体也成对；配子中基因成单，染色体也成单；等位基因分离，同源染色体也分离）。
  • 摩尔根果蝇实验：
    • 材料：果蝇（染色体数少，繁殖快，相对性状明显）。
    • 实验：红眼雌蝇 × 白眼雄蝇 → F₁全为红眼 → F₁雌雄交配 → F₂红眼:白眼 = 3:1，且白眼全为雄性。
    • 解释：控制白眼的基因（w）位于X染色体上，Y染色体上没有等位基因。
    • 结论：基因在染色体上（摩尔根等证明）。
  • 现代解释：一条染色体上有许多基因，基因在染色体上呈线性排列。

伴性遗传

  • 概念：位于性染色体上的基因控制的性状，在遗传上总是与性别相关联。
  • 类型：
    • 伴X显性遗传（如抗维生素D佝偻病）：
      • 特点：女性患者多于男性；男性患者的女儿全患病，母亲患病则儿子可能正常；世代连续遗传。
    • 伴X隐性遗传（如红绿色盲、血友病）：
      • 特点：男性患者多于女性；交叉遗传（男性患者的致病基因来自母亲，传给女儿）；女性患病则其父亲和儿子一定患病。
    • 伴Y遗传（外耳道多毛症）：只传男不传女。
  • 应用：指导优生（如血友病患者生男孩或女孩的建议）；判断基因位置（用正交反交实验）。

第三章 基因的本质

DNA是主要的遗传物质

  • 肺炎链球菌转化实验（格里菲斯）：
    • 体内转化实验：R型（无毒）→小鼠存活；S型（有毒）→小鼠死亡；加热杀死的S型→小鼠存活；R型+加热杀死的S型→小鼠死亡（从死亡小鼠分离出S型）。
    • 结论：加热杀死的S型含有“转化因子”，使R型转化为S型。
  • 艾弗里体外转化实验：
    • 将S型提取物分别用蛋白酶、RNA酶、酯酶、DNA酶处理，然后与R型混合。
    • 只有DNA酶处理组不能使R型转化，其他组都能。
    • 结论：DNA才是使R型产生稳定遗传变化的物质，即DNA是遗传物质。
  • 噬菌体侵染细菌实验（赫尔希、蔡斯）：
    • 原理：用放射性同位素标记法，分别标记T2噬菌体的蛋白质（³⁵S）和DNA（³²P）。
    • 过程：标记噬菌体→侵染未标记细菌→搅拌离心→检测放射性分布。
    • 结果：³⁵S标记组上清液放射性高，沉淀放射性低；³²P标记组沉淀放射性高，上清液放射性低。
    • 结论：DNA进入细菌细胞，蛋白质外壳未进入，因此DNA是遗传物质。
  • RNA也是遗传物质：如烟草花叶病毒（TMV）的RNA侵染实验证明。
  • 结论：绝大多数生物的遗传物质是DNA，所以说DNA是主要的遗传物质。

DNA分子的结构

  • DNA双螺旋结构模型（沃森、克里克）：
    • 由两条反向平行的脱氧核苷酸链盘旋而成。
    • 外侧由磷酸和脱氧核糖交替连接构成基本骨架，内侧是碱基对。
    • 碱基互补配对原则：A（腺嘌呤）与T（胸腺嘧啶）配对，G（鸟嘌呤）与C（胞嘧啶）配对。碱基之间通过氢键连接（A=T两个氢键，G≡C三个氢键）。
  • 结构特点：
    • 稳定性：外侧骨架不变，碱基互补配对固定。
    • 多样性：碱基对排列顺序千变万化，构成了DNA的多样性（4ⁿ种可能）。
    • 特异性：每个DNA分子有特定的碱基排列顺序。
  • DNA中的数量关系：A=T，G=C；A+G=T+C（嘌呤数=嘧啶数）；一条链中(A+T)/(G+C)的比例与互补链相同。

DNA的复制

  • 概念：以亲代DNA为模板合成子代DNA的过程。
  • 时间：有丝分裂间期和减数第一次分裂前的间期。
  • 场所：细胞核（主要）、线粒体、叶绿体。
  • 条件：模板（亲代DNA的两条链）、原料（4种游离的脱氧核苷酸）、能量（ATP）、酶（解旋酶、DNA聚合酶等）。
  • 过程：
    1. 解旋：利用解旋酶断开氢键，两条链解开。
    2. 合成子链：以每条母链为模板，在DNA聚合酶催化下，按照碱基互补配对原则，合成新的子链。
    3. 复旋：新合成的子链与母链盘绕成双螺旋，形成两个子代DNA分子。
  • 特点：边解旋边复制、半保留复制（每个子代DNA分子含一条母链和一条新链）。
  • 意义：将遗传信息从亲代传给子代，保持了遗传信息的连续性。
  • 相关计算：一个DNA分子复制n次，得到2ⁿ个DNA，其中含母链的DNA有2个，母链2条，新链2ⁿ⁺¹-2条。

基因是有遗传效应的DNA片段

  • 基因概念：通常是有遗传效应的DNA片段，是控制生物性状的基本单位。
  • 染色体、DNA、基因、脱氧核苷酸的关系：
    • 染色体由DNA和蛋白质组成，是基因的主要载体。
    • 每个DNA分子上有许多基因，基因在DNA上呈线性排列。
    • 基因的基本组成单位是脱氧核苷酸，不同的碱基排列顺序代表了遗传信息。
  • 遗传信息：基因中脱氧核苷酸的排列顺序（碱基序列）。
  • DNA片段中的非基因区：有些DNA片段没有遗传效应（如间隔序列、调控序列）。

第四章 基因的表达

基因指导蛋白质的合成

  • 转录：
    • 场所：细胞核（主要）。
    • 模板：DNA的一条链（模板链）。
    • 原料：4种核糖核苷酸（A、U、C、G）。
    • 酶：RNA聚合酶（解旋DNA并催化合成RNA）。
    • 过程：DNA解旋，RNA聚合酶催化合成RNA，碱基配对（A-U、T-A、C-G、G-C），合成的RNA释放，DNA双链恢复。
    • 产物：mRNA（信使RNA）、tRNA、rRNA等。
  • 翻译：
    • 场所：核糖体（细胞质）。
    • 模板：mRNA。
    • 原料：20种氨基酸。
    • 运载工具：tRNA（三叶草结构，一端携带氨基酸，一端有反密码子）。
    • 过程：
      1. 起始：mRNA与核糖体结合，起始tRNA（携带甲硫氨酸）识别起始密码子（AUG）。
      2. 延伸：tRNA携带氨基酸进入核糖体，与mRNA上的密码子互补配对，相邻氨基酸通过肽键连接，核糖体沿mRNA移动。
      3. 终止：读到终止密码子（UAA、UAG、UGA），多肽链释放，核糖体解离。
    • 一个mRNA可结合多个核糖体，同时合成多条肽链，提高翻译效率。
  • 遗传密码：mRNA上决定一个氨基酸的三个相邻碱基。密码子具有简并性（一种氨基酸可对应多个密码子）、通用性（几乎适用于所有生物）。

中心法则

  • 经典中心法则（克里克提出）：DNA → DNA（复制） → RNA（转录） → 蛋白质（翻译）。
  • 补充后：
    • RNA → DNA（逆转录，如某些病毒）。
    • RNA → RNA（RNA复制，如RNA病毒）。
    • 蛋白质 → 蛋白质（朊病毒，不常见）。
  • 图解：体现了遗传信息的传递方向，是生物界的普遍规律。

基因表达与性状的关系

  • 直接途径：基因通过控制蛋白质的结构直接控制性状（如血红蛋白基因突变→镰状细胞贫血）。
  • 间接途径：基因通过控制酶的合成来控制代谢过程，进而控制性状（如白化病基因→酪氨酸酶缺乏→黑色素不能合成）。
  • 细胞分化：基因选择性表达的结果，不同细胞中表达的基因种类不同，但DNA相同。
  • 表观遗传：生物体基因的碱基序列保持不变，但基因表达和表型发生可遗传变化（如DNA甲基化、组蛋白修饰）。实例：同卵双胞胎的差异；蜂王和工蜂的差异。
  • 基因与性状的复杂关系：一个基因可能影响多个性状（基因多效性），多个基因也可能控制一个性状（如身高）。环境因素也会影响性状表现。

第五章 基因突变及其他变异

基因突变和基因重组

  • 基因突变：
    • 概念：DNA分子中发生碱基对的替换、增添或缺失，而引起的基因结构的改变。
    • 类型：自发突变（内因）和诱发突变（外因：物理、化学、生物因素）。
    • 特点：普遍性、随机性、不定向性、低频性、多害少利性。
    • 意义：产生新基因的途径，是生物变异的根本来源，为进化提供原材料。
    • 实例：镰状细胞贫血（碱基替换）、癌症（原癌基因和抑癌基因突变）。
  • 基因重组：
    • 概念：生物体进行有性生殖的过程中，控制不同性状的基因重新组合。
    • 类型：自由组合型（减数第一次分裂后期，非同源染色体自由组合）、交叉互换型（减数第一次分裂前期，同源染色体非姐妹染色单体交叉互换）。
    • 意义：产生新的基因型，是生物变异的重要来源，对进化有重要意义。

染色体变异

  • 染色体结构变异：
    • 缺失：染色体片段缺失，导致基因数目减少（如猫叫综合征，5号染色体部分缺失）。
    • 重复：染色体片段重复，导致基因数目增加（如果蝇棒眼）。
    • 倒位：染色体片段颠倒180°，基因顺序改变。
    • 易位：一条染色体的片段移接到另一条非同源染色体上（相互易位），与交叉互换不同（交叉互换发生在同源染色体间）。
  • 染色体数目变异：
    • 类型：个别染色体增减（如21三体综合征）和染色体组倍增增减（如整倍体变异）。
    • 染色体组：细胞中的一组非同源染色体，在形态和功能上各不相同，携带着控制生物生长发育的全部遗传信息。
    • 二倍体：由受精卵发育而来，体细胞含两个染色体组。
    • 多倍体：由受精卵发育而来，体细胞含三个或以上染色体组（如香蕉三倍体、马铃薯四倍体）。特点：茎秆粗壮，叶片大，营养丰富，但结实率低。
    • 单倍体：由配子发育而来，体细胞含本物种配子染色体数（可含一个或多个染色体组）。特点：植株弱小，高度不育。在育种中可快速获得纯合子（经秋水仙素加倍）。
  • 诱发染色体变异的方法：秋水仙素处理萌发的种子或幼苗（抑制纺锤体形成），低温诱导等。

人类遗传病

  • 类型：
    • 单基因遗传病：由一对等位基因控制。常染色体显性（如多指）、常染色体隐性（白化病）、伴X显性（抗维生素D佝偻病）、伴X隐性（红绿色盲）、伴Y遗传。
    • 多基因遗传病：由多对基因控制，易受环境影响，如唇裂、哮喘、原发性高血压。
    • 染色体异常遗传病：由染色体结构或数目异常引起，如21三体综合征（智力低下，特殊面容）、Turner综合征（XO）。
  • 调查人群中的遗传病：最好选择群体中发病率较高的单基因遗传病；调查遗传方式需在患者家系中进行，调查发病率需在人群中随机调查。
  • 监测和预防：
    • 遗传咨询：检查、诊断→分析遗传病传递方式→推算再发风险率→提出建议。
    • 产前诊断：羊水检查、B超、绒毛膜取样、基因检测等。
  • 人类基因组计划：测定人类基因组的全部DNA序列（24条染色体，包括22条常染色体+X+Y）。意义：认识基因结构、功能，为疾病诊治提供依据。

第六章 生物的进化

生物有共同祖先的证据

  • 化石证据：化石是保存在地层中的古代生物的遗体、遗物或遗迹。不同地层中的化石分布，揭示了生物从简单到复杂、从低等到高等的进化顺序。
  • 比较解剖学证据：同源器官（如鸟的翅膀、鲸的鳍、人的上肢，来源相同，功能不同）说明有共同祖先；痕迹器官（如人的阑尾）也提供线索。
  • 胚胎学证据：脊椎动物早期胚胎发育高度相似（如都有咽裂、尾），说明起源于共同祖先。
  • 细胞和分子水平证据：所有生物共用一套遗传密码（DNA、RNA）；细胞色素c等蛋白质的氨基酸序列差异可以推断亲缘关系；DNA杂交技术比较基因相似度。

自然选择与适应的形成

  • 拉马克学说：用进废退、获得性遗传（如长颈鹿的长颈是努力吃高处树叶导致）。
  • 达尔文自然选择学说：
    • 主要内容：过度繁殖、生存斗争、遗传变异、适者生存。
    • 解释：生物普遍存在变异（可遗传）；个体间生存能力不同；适应环境的个体生存并繁殖后代，不适应的淘汰；经过长期自然选择，有利变异积累，形成新类型。
    • 局限性：不能解释遗传变异的本质；对进化的解释局限于个体水平。
  • 适应的形成：适应是自然选择的结果，包括生物对环境的形态、结构、生理、行为等的适应（如保护色、警戒色、拟态）。

种群基因组成的变化与物种形成

  • 种群是生物进化的基本单位：种群是指生活在一定区域的同种生物全部个体的集合。种群基因库是一个种群中全部个体所含有的全部基因。
  • 基因频率和基因型频率：
    • 基因频率 = 某个基因数 / 该基因及其等位基因总数。
    • 基因型频率 = 特定基因型个体数 / 总个体数。
    • 遗传平衡定律（哈代-温伯格定律）：理想种群（无突变、无选择、无迁入迁出、无限大、随机交配）基因频率和基因型频率保持不变。公式：p² + 2pq + q² = 1（p为A基因频率，q为a基因频率）。
  • 种群基因频率的变化：突变和基因重组提供原材料，自然选择导致基因频率定向改变。
  • 物种形成：
    • 隔离：地理隔离（同种生物由于地理障碍不能相遇）和生殖隔离（不同物种之间不能交配或交配后不能产生可育后代）。
    • 物种形成方式：
      • 渐变式：经过长期地理隔离，种群基因库差异积累，形成生殖隔离（如加拉帕戈斯群岛地雀）。
      • 骤变式：通过染色体变异（如多倍体）在短时间内形成新物种，多见于植物。

协同进化与生物多样性

  • 协同进化：不同物种之间、生物与无机环境之间在相互影响中不断进化和发展。例如捕食者与被捕食者、寄生者与宿主、竞争物种间的相互适应；植物与传粉昆虫的相互适应。
  • 生物多样性：
    • 三个层次：遗传多样性、物种多样性、生态系统多样性。
    • 形成原因：协同进化导致物种多样化和生态系统的复杂化。
    • 保护意义：维持生态平衡，为人类提供资源。
