第五章 化工生产中的重要非金属元素

硫及其化合物

硫单质

  • 物理性质：俗称硫黄，黄色晶体，不溶于水，微溶于酒精，易溶于二硫化碳。
  • 化学性质：
    • 与金属反应：\(\ce{Fe + S ->[\text{加热}] FeS}\)、\(\ce{2Cu + S ->[\text{加热}] Cu2S}\)
    • 与非金属反应：\(\ce{S + O2 ->[\text{点燃}] SO2}\)

二氧化硫 (\(\ce{SO2}\))

  • 物理性质：无色有刺激性气味的有毒气体，密度比空气大，易液化，易溶于水。
  • 化学性质：
    • 酸性氧化物：与水反应 \(\ce{SO2 + H2O <=> H2SO3}\)；与碱反应 \(\ce{SO2 + 2NaOH -> Na2SO3 + H2O}\)
    • 还原性：能被 \(\ce{O2}\)、\(\ce{Cl2}\)、\(\ce{KMnO4}\) 等氧化，如 \(\ce{2SO2 + O2 <=>[催化剂][\text{加热}] 2SO3}\)（可逆）
    • 氧化性：与 \(\ce{H2S}\) 反应 \(\ce{SO2 + 2H2S -> 3S v + 2H2O}\)
    • 漂白性：能使品红溶液褪色（加热恢复），用于漂白纸浆、毛、丝等。

三氧化硫 (\(\ce{SO3}\))

  • 无色易挥发晶体，遇水剧烈反应 \(\ce{SO3 + H2O -> H2SO4}\)（放热）。

硫酸 (\(\ce{H2SO4}\))

  • 稀硫酸：具有酸的通性。
  • 浓硫酸的特性：
    • 吸水性：常用作干燥剂（不能干燥碱性气体和还原性气体如 \(\ce{H2S}\)、\(\ce{HI}\)）
    • 脱水性：能将有机物中的氢氧元素按2:1脱去，使蔗糖、纸张等炭化
    • 强氧化性：常温下使 \(\ce{Fe}\)、\(\ce{Al}\) 钝化；加热时能与大多数金属反应，如 \(\ce{Cu + 2H2SO4(浓) ->[\text{加热}] CuSO4 + SO2 ^ + 2H2O}\)；也能与非金属反应，如 \(\ce{C + 2H2SO4(浓) ->[\text{加热}] CO2 ^ + 2SO2 ^ + 2H2O}\)

氮及其化合物

氮气 (\(\ce{N2}\))

  • 无色无味气体，约占空气体积的78%，化学性质稳定（含 \(\ce{N≡N}\) 三键）。
  • 一定条件下反应：\(\ce{N2 + O2 ->[放电] 2NO}\)，\(\ce{N2 + 3H2 ->[高温、高压、催化剂] 2NH3}\)。

一氧化氮 (\(\ce{NO}\))

  • 无色无味有毒气体，不溶于水，在空气中易被氧化：\(\ce{2NO + O2 -> 2NO2}\)。

二氧化氮 (\(\ce{NO2}\))

  • 红棕色有刺激性气味气体，易溶于水：\(\ce{3NO2 + H2O -> 2HNO3 + NO}\)。

氨 (\(\ce{NH3}\))

  • 物理性质：无色刺激性气味气体，密度比空气小，极易溶于水（1:700），易液化。
  • 化学性质：
    • 与水反应：\(\ce{NH3 + H2O <=> NH3·H2O <=> NH4+ + OH-}\)（可逆，氨水显弱碱性）
    • 与酸反应：\(\ce{NH3 + HCl -> NH4Cl}\)（白烟）
    • 催化氧化：\(\ce{4NH3 + 5O2 ->[Pt][\text{加热}] 4NO + 6H2O}\)
    • 实验室制法：\(\ce{2NH4Cl + Ca(OH)2 ->[\text{加热}] CaCl2 + 2NH3 ^ + 2H2O}\)

铵盐

  • 易溶于水，受热易分解：\(\ce{NH4Cl ->[\text{加热}] NH3 ^ + HCl ^}\)，\(\ce{NH4HCO3 ->[\text{加热}] NH3 ^ + CO2 ^ + H2O}\)
  • 与碱反应：\(\ce{NH4+ + OH- -> NH3 ^ + H2O}\)（检验铵根）

硝酸 (\(\ce{HNO3}\))

  • 物理性质：无色易挥发液体，有刺激性气味。
  • 化学性质：
    • 不稳定性：\(\ce{4HNO3(浓) ->[光照或\text{加热}] 4NO2 ^ + O2 ^ + 2H2O}\)，应保存在棕色瓶中
    • 强氧化性：浓硝酸与大多数金属反应，不生成 \(\ce{H2}\)，如 \(\ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2 ^ + 2H2O}\)，\(\ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO ^ + 4H2O}\)；常温下使 \(\ce{Fe}\)、\(\ce{Al}\) 钝化

硅及其化合物

硅单质

  • 物理性质：灰黑色固体，有金属光泽，硬而脆，半导体材料。
  • 化学性质：常温下只与 \(\ce{F2}\)、\(\ce{HF}\) 和强碱反应：\(\ce{Si + 4HF -> SiF4 ^ + 2H2 ^}\)，\(\ce{Si + 2NaOH + H2O -> Na2SiO3 + 2H2 ^}\)。

二氧化硅 (\(\ce{SiO2}\))

  • 物理性质：坚硬难熔固体，不溶于水。
  • 化学性质：
    • 与氢氟酸反应（特性）：\(\ce{SiO2 + 4HF -> SiF4 ^ + 2H2O}\)（用于雕刻玻璃）
    • 与碱反应：\(\ce{SiO2 + 2NaOH -> Na2SiO3 + H2O}\)（碱液不用玻璃塞）
    • 高温下与碱性氧化物反应：\(\ce{SiO2 + CaO ->[\text{高温}] CaSiO3}\)

硅酸 (\(\ce{H2SiO3}\))

  • 难溶于水的弱酸，由硅酸盐与酸反应制得：\(\ce{Na2SiO3 + 2HCl -> 2NaCl + H2SiO3 v}\)。

硅酸盐

  • 硅酸钠 (\(\ce{Na2SiO3}\))：俗称泡花碱，水溶液俗称水玻璃，可作黏合剂、防火剂。
  • 传统硅酸盐材料：陶瓷、玻璃、水泥。

第六章 化学反应与能量

化学反应与热能

放热反应与吸热反应

  • 放热反应：反应物总能量 > 生成物总能量，如中和、燃烧、金属与酸、大多数化合反应。
  • 吸热反应：反应物总能量 < 生成物总能量，如大多数分解反应、\(\ce{C}\) 与 \(\ce{CO2}\) 反应、\(\ce{Ba(OH)2·8H2O}\) 与 \(\ce{NH4Cl}\) 反应。

反应热与焓变

  • 恒压条件下，反应的热效应等于焓变 \(\Delta H\)，单位 kJ/mol。
  • \(\Delta H\) = 生成物总能量 – 反应物总能量 = 反应物键能总和 – 生成物键能总和。
  • \(\Delta H < 0\) 为放热，\(\Delta H > 0\) 为吸热。

热化学方程式

  • 需注明物质状态和反应热，如 \(\ce{H2(g) + 1/2O2(g) = H2O(l)}\) \(\Delta H = -285.8\ \text{kJ/mol}\)。

中和热

  • 稀溶液中，强酸强碱中和生成 1 mol \(\ce{H2O}\) 的反应热，约为 –57.3 kJ/mol。

燃烧热

  • 101 kPa 时，1 mol 纯物质完全燃烧生成稳定氧化物时所放出的热量。

化学反应与电能

原电池

  • 定义：将化学能转化为电能的装置。
  • 构成条件：两个活泼性不同的电极、电解质溶液、形成闭合回路、自发的氧化还原反应。
  • 工作原理：
    • 负极：发生氧化反应（较活泼金属，电子流出）
    • 正极：发生还原反应（较不活泼金属或导电非金属，电子流入）
    • 溶液中阳离子向正极移动，阴离子向负极移动
  • 举例：锌铜原电池，负极 \(\ce{Zn -> Zn^{2+} + 2e^-}\)，正极 \(\ce{2H+ + 2e^- -> H2 ^}\)。

化学电源

  • 一次电池：不能充电，如普通干电池。
  • 二次电池：可充电，如铅蓄电池。
  • 燃料电池：将燃料与氧化剂的化学能直接转化为电能，如氢氧燃料电池，总反应 \(\ce{2H2 + O2 -> 2H2O}\)。

金属的腐蚀与防护

  • 电化学腐蚀：钢铁的析氢腐蚀（酸性环境）和吸氧腐蚀（中性或碱性环境）。
  • 防护方法：覆盖保护层、改变内部结构、电化学保护法（牺牲阳极的阴极保护法、外加电流的阴极保护法）。

化学反应的速率与限度

化学反应速率

  • 定义：单位时间内反应物浓度的减少或生成物浓度的增加，单位 mol/(L·s) 或 mol/(L·min)。公式 \(v = \frac{\Delta c}{\Delta t}\)。
  • 影响因素：
    • 内因：反应物本身的性质。
    • 外因：浓度（增大浓度，速率加快）、压强（对有气体参加的反应，增大压强，速率加快）、温度（升高温度，速率加快）、催化剂（正催化剂加快反应速率）、接触面积等。

可逆反应与化学平衡

  • 可逆反应：在同一条件下既能正向进行又能逆向进行的反应。
  • 化学平衡状态：正反应速率与逆反应速率相等，反应物和生成物浓度不再改变的状态。特征：逆、等、动、定、变。

化学平衡的移动

  • 勒夏特列原理：如果改变影响平衡的条件之一（浓度、压强、温度），平衡将向着减弱这种改变的方向移动。催化剂不改变平衡移动。

第七章 有机化合物

有机物的结构特点

  • 有机化合物：含有碳元素的化合物（但 \(\ce{CO}\)、\(\ce{CO2}\)、碳酸盐等除外）。
  • 碳原子的成键特点：碳原子易形成4个共价键，可以形成单键、双键、三键，碳原子间可连接成链状或环状。
  • 同分异构现象：化合物具有相同的分子式，但结构不同的现象。同分异构体性质有差异。
  • 官能团：决定有机物特性的原子或原子团，如碳碳双键、羟基（-OH）、羧基（-COOH）、酯基（-COO-）等。

甲烷与烷烃

甲烷 (\(\ce{CH4}\))

  • 物理性质：无色无味气体，难溶于水，密度比空气小。
  • 化学性质：
    • 氧化反应：燃烧 \(\ce{CH4 + 2O2 ->[\text{点燃}] CO2 + 2H2O}\)（淡蓝色火焰）
    • 取代反应：光照下与氯气发生取代，产物混合物：\(\ce{CH3Cl}\)、\(\ce{CH2Cl2}\)、\(\ce{CHCl3}\)、\(\ce{CCl4}\)、\(\ce{HCl}\)
    • 分解反应：\(\ce{CH4 ->[\text{高温}] C + 2H2}\)

烷烃

  • 通式 \(\ce{C_nH_{2n+2}}\) (n≥1)。
  • 性质：熔沸点随碳原子数增加而升高，均难溶于水，化学性质稳定，可发生燃烧和取代反应。

乙烯与烯烃

乙烯 (\(\ce{C2H4}\))

  • 物理性质：无色稍有气味的气体，难溶于水，密度比空气略小。
  • 化学性质：
    • 氧化反应：燃烧火焰明亮有黑烟；能使酸性 \(\ce{KMnO4}\) 溶液褪色
    • 加成反应：与 \(\ce{Br2}\) 的 \(\ce{CCl4}\) 溶液反应 \(\ce{CH2=CH2 + Br2 -> CH2BrCH2Br}\)；与 \(\ce{H2O}\) 加成制乙醇；与 \(\ce{HCl}\) 加成制氯乙烷
    • 加聚反应：\(\ce{nCH2=CH2 ->[催化剂] [CH2-CH2]_n}\)

烯烃

  • 含碳碳双键的不饱和烃，通式 \(\ce{C_nH_{2n}}\) (n≥2)，能发生加成、加聚、氧化等反应。

苯及其同系物

苯 (\(\ce{C6H6}\))

  • 物理性质：无色有特殊气味的液体，密度比水小，不溶于水，易挥发，有毒。
  • 分子结构：平面正六边形，键介于单双键之间，所有原子共平面。
  • 化学性质：
    • 氧化反应：燃烧火焰明亮有浓烟；不能使酸性 \(\ce{KMnO4}\) 褪色
    • 取代反应：卤代 \(\ce{C6H6 + Br2 ->[FeBr3] C6H5Br + HBr}\)；硝化 \(\ce{C6H6 + HNO3(浓) ->[浓H2SO4][50-60℃] C6H5NO2 + H2O}\)
    • 加成反应：\(\ce{C6H6 + 3H2 ->[Ni][\text{加热}] C6H12}\)

苯的同系物

  • 烷基苯，通式 \(\ce{C_nH_{2n-6}}\) (n≥6)，侧链能被酸性 \(\ce{KMnO4}\) 氧化（如甲苯使 \(\ce{KMnO4}\) 褪色）。

乙醇、乙酸、基本营养物质

乙醇 (\(\ce{C2H5OH}\))

  • 物理性质：无色特殊香味的液体，易挥发，与水任意比互溶。
  • 化学性质：
    • 与钠反应：\(\ce{2C2H5OH + 2Na -> 2C2H5ONa + H2 ^}\)
    • 氧化反应：燃烧 \(\ce{C2H5OH + 3O2 ->[\text{点燃}] 2CO2 + 3H2O}\)；催化氧化 \(\ce{2C2H5OH + O2 ->[Cu或Ag][\text{加热}] 2CH3CHO + 2H2O}\)
    • 酯化反应：与乙酸反应生成乙酸乙酯

乙酸 (\(\ce{CH3COOH}\))

  • 物理性质：强烈刺激性气味的液体，易溶于水。
  • 化学性质：
    • 酸性：具有酸的通性，能使紫色石蕊变红
    • 酯化反应：\(\ce{CH3COOH + C2H5OH <=>[浓H2SO4][\text{加热}] CH3COOC2H5 + H2O}\)（可逆）

基本营养物质

  • 糖类：单糖（葡萄糖、果糖）、二糖（蔗糖、麦芽糖）、多糖（淀粉、纤维素）。葡萄糖能发生银镜反应，淀粉遇碘变蓝。
  • 油脂：高级脂肪酸甘油酯，碱性条件下水解生成高级脂肪酸盐（肥皂）和甘油（皂化反应）。
  • 蛋白质：由氨基酸脱水缩合而成，盐析（可逆）与变性（不可逆）的区别；蛋白质遇浓硝酸变黄（颜色反应）。

第八章 化学与可持续发展

自然资源的开发利用

金属矿物的开发利用

  • 热分解法（不活泼金属如 \(\ce{Hg}\)、\(\ce{Ag}\)）
  • 热还原法（中等活泼金属如 \(\ce{Fe}\)、\(\ce{Cu}\)）
  • 电解法（活泼金属如 \(\ce{Na}\)、\(\ce{Mg}\)、\(\ce{Al}\)）

海水资源的开发利用

  • 海水制盐、海水提溴（空气吹出法）、海水提镁、海水淡化。

化石燃料的综合利用

  • 煤：干馏（化学变化）、气化、液化。
  • 石油：分馏（物理变化）、裂化、裂解、催化重整。

化学品的合理使用

  • 化肥、农药的合理施用：过量使用会导致土壤酸化、水体富营养化。
  • 常见药物的正确使用：解热镇痛药（阿司匹林）、抗生素（青霉素需皮试）、抗酸药（胃舒平）。
  • 食品添加剂：着色剂、增味剂、防腐剂、抗氧化剂等，合理使用可改善食品品质。

环境保护与绿色化学

环境问题

  • 大气污染：\(\ce{SO2}\)、\(\ce{NOx}\)、可吸入颗粒物等导致酸雨、光化学烟雾、温室效应。
  • 水体污染：富营养化（赤潮、水华）、重金属污染。
  • 土壤污染：化肥、农药、重金属等。

绿色化学

  • 核心：从源头上减少或消除对环境的污染。
  • 原则：原子经济性（原子利用率100%）、高选择性、使用无毒无害原料等。
  • 示例：乙烯直接氧化法制环氧乙烷（传统氯醇法有污染）。
