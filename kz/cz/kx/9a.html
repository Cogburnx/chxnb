第一章 物质及其变化

1.1 物质的变化

  • 物理变化与化学变化：
    • 物理变化：没有新物质生成的变化。如状态改变、形状改变、溶解等。
    • 化学变化：有新物质生成的变化，常伴随发光、发热、变色、产生气体、沉淀等现象。如燃烧、铁生锈、食物腐败等。
  • 物理性质与化学性质：
    • 物理性质：不需要发生化学变化就能表现出来的性质，如颜色、状态、气味、密度、熔点、沸点、溶解性等。
    • 化学性质：在化学变化中表现出来的性质，如可燃性、稳定性、氧化性、还原性、酸性、碱性等。

1.2 物质的酸碱性

  • 酸性溶液与碱性溶液：含有较多H⁺的溶液显酸性，含有较多OH⁻的溶液显碱性。
  • 酸碱指示剂：
    • 紫色石蕊试液：遇酸变红，遇碱变蓝。
    • 无色酚酞试液：遇酸不变色，遇碱变红。
  • pH与酸碱度：pH范围0-14，pH=7中性，pH<7酸性，pH>7碱性。pH越小酸性越强，越大碱性越强。
  • pH的测定：用pH试纸（标准比色卡）或pH计。

1.3 常见的酸

  • 盐酸（HCl）：无色刺激性气味液体，工业盐酸呈黄色（含Fe³⁺）。能与活泼金属反应生成氢气，与金属氧化物反应生成盐和水，与碱反应生成盐和水，与盐反应生成新酸和新盐（复分解反应）。
  • 硫酸（H₂SO₄）：无色油状液体，浓硫酸有脱水性、吸水性（作干燥剂）、强腐蚀性。稀释浓硫酸时，应将浓硫酸沿器壁缓慢注入水中，并不断搅拌。稀硫酸性质与盐酸相似。
  • 硝酸（HNO₃）：强氧化性，与金属反应一般不生成氢气。
  • 酸的共性：①使指示剂变色；②与活泼金属→盐+氢气；③与金属氧化物→盐+水；④与碱→盐+水（中和反应）；⑤与某些盐→新酸+新盐。

1.4 常见的碱

  • 氢氧化钠（NaOH）：俗称烧碱、火碱、苛性钠，白色固体，易溶于水，强腐蚀性，易潮解（可作干燥剂）。能与酸反应，与某些非金属氧化物反应（如CO₂、SO₂）。
  • 氢氧化钙（Ca(OH)₂）：俗称熟石灰、消石灰，白色粉末，微溶于水，腐蚀性较弱。用于建筑、改良酸性土壤、配制波尔多液等。
  • 碱的共性：①使指示剂变色（紫色石蕊变蓝，无色酚酞变红）；②与酸反应生成盐和水；③与某些非金属氧化物反应生成盐和水；④与某些盐反应生成新碱和新盐。

1.5 酸和碱之间的反应

  • 中和反应：酸和碱作用生成盐和水的反应。通式：酸 + 碱 → 盐 + 水。实质是H⁺ + OH⁻ = H₂O。
  • 中和反应的应用：①改良酸性土壤（用熟石灰）；②处理工业废水（酸碱中和）；③医药（胃酸过多用氢氧化铝、碳酸氢钠等）；④调节溶液的酸碱性。
  • 中和反应的实验探究：用酚酞作指示剂，证明碱与酸反应后碱性消失。

1.6 几种重要的盐

  • 盐的组成：由金属离子（或铵根离子）和酸根离子构成的化合物。
  • 常见盐的用途：
    • 氯化钠（NaCl）：调味品、防腐剂、制生理盐水。
    • 碳酸钠（Na₂CO₃）：俗称纯碱、苏打，用于玻璃、造纸、洗涤剂等。
    • 碳酸氢钠（NaHCO₃）：俗称小苏打，发酵粉、治疗胃酸过多。
    • 碳酸钙（CaCO₃）：大理石、石灰石，建筑材料、补钙剂。
  • 盐的化学性质：①与金属反应生成新盐和新金属；②与酸反应生成新盐和新酸；③与碱反应生成新盐和新碱；④与盐反应生成两种新盐（复分解反应条件：生成物有沉淀、气体或水）。
  • 复分解反应：两种化合物互相交换成分，生成另外两种化合物的反应。通式：AB + CD → AD + CB。

1.7 物质的鉴别与推断

  • 常见离子的检验：
    • H⁺：紫色石蕊变红；pH试纸；活泼金属产生氢气；碳酸盐产生CO₂。
    • OH⁻：紫色石蕊变蓝；无色酚酞变红；pH试纸；Fe³⁺溶液产生红褐色沉淀。
    • Cl⁻：AgNO₃溶液和稀硝酸，生成白色沉淀（AgCl）。
    • SO₄²⁻：BaCl₂或Ba(NO₃)₂溶液和稀硝酸，生成白色沉淀（BaSO₄）。
    • CO₃²⁻：加稀盐酸，产生使澄清石灰水变浑浊的气体。
  • 物质的推断方法：根据实验现象、颜色、气味等特征，结合物质的性质进行推理，确定未知物质。

第二章 物质转化与材料利用

2.1 金属材料

  • 金属的物理性质：有光泽、导电性、导热性、延展性、常温下为固体（汞除外）。
  • 合金：在金属中加热熔合某些金属或非金属制成的具有金属特性的物质。合金硬度一般比组分金属大，熔点较低。如钢（铁碳合金）、黄铜（铜锌合金）、铝合金等。
  • 金属材料的选择：根据用途综合考虑性质、价格、资源、环境等因素。

2.2 金属的化学性质

  • 金属与氧气反应：大多数金属能与氧气反应生成金属氧化物。如镁在空气中燃烧发出耀眼白光，铁在氧气中燃烧火星四射，铝表面形成致密氧化膜。
  • 金属与酸反应：活泼金属（如Mg、Al、Zn、Fe）能与稀盐酸、稀硫酸反应生成氢气，不活泼金属（如Cu、Ag）不反应。铁与酸反应生成亚铁盐（Fe²⁺）。
  • 金属与盐溶液反应：活泼金属能将不活泼金属从其盐溶液中置换出来。如Fe + CuSO₄ = FeSO₄ + Cu（湿法炼铜）。
  • 金属活动性顺序：K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au。从左到右活动性减弱。①排在氢前面的金属能置换出酸中的氢；②排在前面的金属能把排在后面的金属从其盐溶液中置换出来。

2.3 金属的冶炼

  • 炼铁原理：高炉中用一氧化碳还原铁矿石（主要成分Fe₂O₃）。反应：Fe₂O₃ + 3CO \(\xrightarrow{\text{高温}}\) 2Fe + 3CO₂。
  • 铁的锈蚀与防护：铁生锈条件（氧气、水）。防护方法：①保持干燥；②涂保护层（油漆、电镀）；③制成合金（不锈钢）；④牺牲阳极保护法等。
  • 金属资源的保护：防止金属腐蚀；回收利用废旧金属；合理开采矿物；寻找金属代用品。

2.4 有机物与材料

  • 有机物：含碳的化合物（除CO、CO₂、碳酸盐等）。如甲烷（CH₄）、乙醇（C₂H₅OH）、葡萄糖（C₆H₁₂O₆）等。
  • 有机合成材料：塑料、合成纤维、合成橡胶。三大合成材料：塑料、合成橡胶、合成纤维。
  • 复合材料：由两种或两种以上不同材料复合而成，具有更优良的性能。如玻璃钢、碳纤维复合材料等。

2.5 物质的转化

  • 物质的转化规律：
    • 金属 → 金属氧化物 → 碱 → 盐（如Ca→CaO→Ca(OH)₂→CaCO₃）
    • 非金属 → 非金属氧化物 → 酸 → 盐（如C→CO₂→H₂CO₃→Na₂CO₃）
  • 化合反应、分解反应、置换反应、复分解反应四种基本反应类型。
  • 氧化还原反应：物质得到氧（被氧化）或失去氧（被还原）的反应。氧化剂和还原剂。

2.6 材料的发展与利用

  • 材料发展的历史：石器时代→青铜时代→铁器时代→信息时代（硅材料）→新材料时代。
  • 新材料的应用：纳米材料、超导材料、功能高分子材料、绿色能源材料等。
  • 材料与可持续发展：倡导绿色化学，减少废弃物，使用可再生资源。

第三章 能量的转化与守恒

3.1 能量及其形式

  • 能量的形式：机械能、内能、电能、化学能、光能、核能等。
  • 能量可以从一种形式转化为另一种形式：如电动机将电能转化为机械能，电池将化学能转化为电能，光合作用将光能转化为化学能。

3.2 机械能

  • 动能：物体由于运动而具有的能量。大小与质量、速度有关：\(E_k = \frac{1}{2}mv^2\)。
  • 势能：重力势能（与质量、高度有关：\(E_p = mgh\)）和弹性势能（与形变程度有关）。
  • 机械能守恒：只有重力或弹力做功时，动能和势能相互转化，总和不变。

3.3 功和功率

  • 功：力与物体在力的方向上移动距离的乘积。公式 \(W = Fs\)，单位焦耳（J）。做功的两个必要因素：作用在物体上的力，物体在力的方向上通过的距离。
  • 功率：单位时间内所做的功。公式 \(P = \frac{W}{t}\)，单位瓦特（W）。表示做功的快慢。
  • 常见功率估算：人上楼约几百瓦，汽车几十千瓦。

3.4 简单机械

  • 杠杆：在力的作用下能绕固定点转动的硬棒。五要素：支点、动力、阻力、动力臂、阻力臂。杠杆平衡条件：\(F_1l_1 = F_2l_2\)。分类：省力杠杆（l₁>l₂）、费力杠杆（l₁
  • 滑轮：
    • 定滑轮：实质是等臂杠杆，不省力但改变力的方向。
    • 动滑轮：实质是动力臂为阻力臂2倍的杠杆，省一半力，不改变方向。
    • 滑轮组：既能省力又能改变方向，省力情况：\(F = \frac{1}{n}G\)（n为承担重物绳子的段数）。
  • 机械效率：有用功与总功的比值，\(\eta = \frac{W_{\text{有}}}{W_{\text{总}}} \times 100\%\)。小于1。

3.5 动能和势能

  • 动能与势能的转化：如单摆、滚摆、过山车等。不计阻力时机械能守恒。
  • 实验探究：探究影响动能大小的因素（控制变量法，通过木块被推距离比较动能）。

3.6 电能

  • 电功：电流所做的功。公式 \(W = UIt\)，单位焦耳（J），常用千瓦时（kWh），1kWh=3.6×10⁶J。电能表测量电功。
  • 电功率：电流在单位时间内所做的功。公式 \(P = UI = \frac{W}{t}\)，单位瓦特（W）。额定电压、额定功率。

3.7 电功率

  • 电功率的计算：\(P = UI = I^2R = \frac{U^2}{R}\)（适用于纯电阻电路）。
  • 实际功率与额定功率：用电器实际电压等于额定电压时实际功率等于额定功率；实际电压低于额定电压时实际功率小于额定功率，可能无法正常工作。
  • 测量小灯泡的电功率：伏安法，通过调节滑动变阻器测不同电压下的电流，计算电功率。

3.8 焦耳定律

  • 电流的热效应：电流通过导体时产生热量的现象。焦耳定律：电流通过导体产生的热量与电流的平方、导体的电阻、通电时间成正比。\(Q = I^2Rt\)。
  • 电热的应用与防止：电热器（电饭煲、电熨斗）利用电热；散热风扇、散热片防止过热。

3.9 能量的转化与守恒

  • 能量守恒定律：能量既不会凭空消灭，也不会凭空产生，它只会从一种形式转化为其他形式，或者从一个物体转移到其他物体，而在转化和转移的过程中，能量的总量保持不变。
  • 能量转化和转移的方向性：实际过程中，能量的转化和转移是有方向性的。如内能无法全部用来做功而不产生其他影响（热力学第二定律）。
  • 能源与可持续发展：常规能源（煤、石油、天然气）不可再生；新能源（太阳能、风能、核能、地热能等）清洁、可再生。提高能源利用率，开发新能源。

第四章 代谢与平衡

4.1 食物与营养

  • 七大营养素：糖类、脂肪、蛋白质、水、无机盐、维生素、膳食纤维。
  • 供能物质：糖类（主要供能）、脂肪（储备能源）、蛋白质（也可供能）。
  • 维生素与无机盐：缺乏维生素A→夜盲症；维生素C→坏血病；维生素D→佝偻病；缺钙→骨质疏松；缺铁→贫血；缺碘→甲状腺肿大。

4.2 消化与吸收

  • 消化系统的组成：消化道（口腔、咽、食道、胃、小肠、大肠、肛门）和消化腺（唾液腺、肝脏、胰腺、胃腺、肠腺）。
  • 主要消化液及其作用：
    • 唾液：淀粉酶，初步分解淀粉为麦芽糖。
    • 胃液：胃蛋白酶，初步分解蛋白质。
    • 胆汁（肝脏）：乳化脂肪，无消化酶。
    • 胰液和肠液：含多种酶，彻底消化糖类、脂肪、蛋白质。
  • 吸收的主要场所：小肠（结构特点：长、皱襞绒毛、毛细血管丰富）。

4.3 体内物质的运输

  • 血液的成分：血浆（运载血细胞、运输养料和废物）、血细胞（红细胞、白细胞、血小板）。
  • 血管类型：动脉（管壁厚、血流快）、静脉（管壁薄、有静脉瓣）、毛细血管（连通动脉静脉，物质交换场所）。
  • 心脏结构：四个腔（左心房、左心室、右心房、右心室），左心室壁最厚。血液循环途径：体循环（左心室→主动脉→全身毛细血管→上下腔静脉→右心房）和肺循环（右心室→肺动脉→肺部毛细血管→肺静脉→左心房）。
  • 血液成分变化：体循环：动脉血变静脉血；肺循环：静脉血变动脉血。

4.4 能量的获得

  • 呼吸作用：细胞内的有机物在氧的参与下被分解成二氧化碳和水，同时释放能量的过程。场所：细胞质基质、线粒体。反应式：\(C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O + 能量\)。
  • 无氧呼吸：在缺氧条件下，细胞将有机物分解为不彻底的氧化产物，释放少量能量。如乳酸发酵（动物、乳酸菌）、酒精发酵（植物、酵母菌）。
  • 能量的利用：用于维持体温、肌肉收缩、神经传导、合成代谢等。

4.5 体内物质的动态平衡

  • 排泄途径：呼吸系统（CO₂、水）、皮肤（汗液）、泌尿系统（尿液）。
  • 泌尿系统的组成：肾脏（形成尿液）、输尿管、膀胱、尿道。
  • 尿液的形成：肾小球的滤过作用（形成原尿）和肾小管的重吸收作用（形成尿液）。原尿与血浆相比无大分子蛋白质；尿液与原尿相比无葡萄糖、水减少等。
  • 水盐平衡：通过尿量调节、出汗等维持体内水盐平衡。抗利尿激素（ADH）调节水平衡。

4.6 代谢的多样性

  • 新陈代谢的类型：
    • 同化作用：合成有机物、储存能量（自养型：光合作用、化能合成；异养型：捕食、寄生、腐生）。
    • 异化作用：分解有机物、释放能量（需氧型、厌氧型）。
  • 微生物的代谢特点：如乳酸菌是异养厌氧型，酵母菌是异养兼性厌氧型，硝化细菌是自养需氧型。
  • 代谢与健康：代谢紊乱导致疾病（糖尿病、痛风等）。
