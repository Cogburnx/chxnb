第一章 水和水的溶液

1.1 地球上的水

  • 水的分布：地球表面约71%被水覆盖，但淡水仅占2.5%，可利用的淡水不足1%。
  • 水循环：蒸发（蒸腾）→水汽输送→降水→径流→海洋，使水资源不断更新。
  • 水资源分布不均：我国南方多、北方少，夏秋多、冬春少，需跨流域调水（如南水北调）。

1.2 水的组成

  • 电解水实验：正极产生氧气，负极产生氢气，体积比V(H₂):V(O₂)=2:1。证明水由氢元素和氧元素组成，化学式H₂O。
  • 水的物理性质：无色无味透明液体，4℃时密度最大（1g/cm³），冰的密度小于水（0.9g/cm³）。

1.3 水的浮力

  • 浮力：浸在液体（或气体）中的物体受到向上的托力，方向竖直向上。
  • 阿基米德原理：浸在液体中的物体受到向上的浮力，浮力大小等于物体排开的液体所受的重力。公式：\(F_{浮} = G_{排} = \rho_{液} g V_{排}\)。
  • 物体的浮沉条件：
    • 上浮：\(F_{浮} > G\)，\(\rho_{物} < \rho_{液}\)。
    • 悬浮：\(F_{浮} = G\)，\(\rho_{物} = \rho_{液}\)。
    • 下沉：\(F_{浮} < G\)，\(\rho_{物} > \rho_{液}\)。
  • 浮力的应用：轮船（空心增大排液体积）、潜水艇（改变自重）、密度计（漂浮原理，刻度上小下大）。

1.4 物质在水中的分散状况

  • 溶液：一种或几种物质分散到另一种物质中，形成均一、稳定的混合物。特征：均一性、稳定性。溶质（被溶解的物质）、溶剂（能溶解其他物质的物质）。
  • 悬浊液：固体小颗粒悬浮在液体中，不均一、不稳定（如泥水）。
  • 乳浊液：小液滴分散在液体中，不均一、不稳定（如油水混合物）。

1.5 物质的溶解

  • 溶解度：在一定温度下，某固态物质在100g溶剂中达到饱和状态时所溶解的质量。影响因素：温度（大多数固体随温度升高而增大，熟石灰随温度升高而减小）、压强（气体溶解度随压强增大而增大，随温度升高而减小）。
  • 饱和溶液与不饱和溶液：一定温度下，不能再溶解某种溶质的溶液为饱和溶液，反之为不饱和溶液。相互转化：改变温度、加溶质、蒸发溶剂等。
  • 溶质质量分数：\(\omega = \frac{m_{溶质}}{m_{溶液}} \times 100\%\)。可用于计算溶液的配制和稀释。

1.6 物质的分离

  • 结晶：从溶液中析出固体的过程。方法：蒸发结晶（适用于溶解度受温度影响不大的物质，如NaCl）、冷却热饱和溶液（适用于溶解度受温度影响大的物质，如KNO₃）。
  • 过滤：分离固体和液体混合物。一贴、二低、三靠（滤纸紧贴漏斗内壁；液面低于滤纸边缘、滤纸边缘低于漏斗边缘；烧杯口紧靠玻璃棒、玻璃棒紧靠三层滤纸、漏斗下端紧靠烧杯内壁）。
  • 蒸馏：分离沸点不同的液体混合物。利用加热使液体汽化，再冷凝液化。

1.7 水资源的利用、开发和保护

  • 水资源危机：淡水短缺、水污染（工业废水、农业化肥、生活污水）。
  • 保护措施：节约用水、防治水污染、合理开发地下水、跨流域调水、海水淡化等。

第二章 天气与气候

2.1 大气层

  • 大气分层：对流层（天气现象）、平流层（臭氧层、飞机飞行）、中间层、暖层、散逸层。对流层高度随纬度、季节变化（低纬厚、夏季厚）。
  • 大气的垂直温度变化：对流层气温随高度增加而降低（每升高100m降0.6℃），平流层随高度增加而升高。

2.2 气温

  • 气温的测量：百叶箱（离地面1.5m，避免阳光直射），单位℃。一天中最高温出现在14时左右，最低温出现在日出前后。年较差：最热月与最冷月平均气温差。

2.3 大气的压强

  • 大气压的存在：马德堡半球实验证明。大气压随高度增加而减小，随天气变化（晴天气压高，阴雨天气压低）。
  • 大气压的测量：托里拆利实验，标准大气压=760mm Hg=1.013×10⁵Pa。
  • 大气压与生活：吸盘、吸管、抽水机、高压锅等利用大气压。

2.4 风和降水

  • 风：空气的水平运动。风向（风吹来的方向），风速（风级）。
  • 降水形成条件：空气上升冷却、水汽充足、有凝结核。降水形式：雨、雪、冰雹等。
  • 湿度与降水：相对湿度表示空气中水汽的饱和程度，干湿球温度计测量。

2.5 天气预报

  • 天气图与符号：高压中心（晴朗），低压中心（阴雨），冷锋（降温降水），暖锋（升温降水）。
  • 常用天气符号：晴、多云、阴、小雨、中雨、大雨、雷阵雨、雾、雪等。

2.6 气候和影响气候的因素

  • 气候：某一地区多年天气的平均状况。影响气候的因素：纬度位置（太阳辐射）、海陆位置（海洋性气候与大陆性气候）、地形（海拔每升高1000m气温下降6℃，迎风坡降水多）、季风（冬季风寒冷干燥，夏季风温暖湿润）。

2.7 我国的气候特征与主要气象灾害

  • 气候特征：季风气候显著（东部），大陆性气候强（西部），气候复杂多样。
  • 主要气象灾害：寒潮（冬季）、台风（夏秋）、洪涝、干旱、沙尘暴等。

第三章 生命活动的调节

3.1 植物生命活动的调节

  • 向性运动：植物受外界刺激引起的定向运动。向光性（茎向光弯曲）、向地性（根向地、茎背地）、向水性等。原因是生长素分布不均匀。
  • 植物激素：生长素（促进细胞伸长，高浓度抑制生长）、赤霉素、细胞分裂素、脱落酸、乙烯等，调节生长发育。

3.2 人体的激素调节

  • 内分泌腺：垂体（生长激素）、甲状腺（甲状腺激素）、肾上腺、胰岛（胰岛素）、性腺等。
  • 激素的作用：
    • 生长激素：幼年不足→侏儒症；过多→巨人症；成年过多→肢端肥大症。
    • 甲状腺激素：促进发育，提高神经系统兴奋性。幼年不足→呆小症；过多→甲亢。
    • 胰岛素：降低血糖。分泌不足→糖尿病（注射胰岛素）。

3.3 神经调节

  • 神经元：神经系统结构和功能的基本单位，包括细胞体、树突、轴突。功能：接受刺激，产生并传导兴奋。
  • 反射与反射弧：反射是神经调节的基本方式，反射弧包括感受器→传入神经→神经中枢→传出神经→效应器。
  • 中枢神经系统：脑（大脑、小脑、脑干）和脊髓。大脑皮层是最高级中枢，小脑维持平衡，脑干调节心跳、呼吸等基本生命活动。

3.4 动物的行为

  • 先天性行为：生来就有，由遗传决定（如婴儿吮吸、蜜蜂采蜜）。
  • 后天学习行为：在遗传基础上，通过环境因素和学习获得（如小狗算数、猴子骑车）。
  • 动物行为的意义：适应环境，生存繁衍。

3.5 体温调节

  • 恒温动物与变温动物：鸟类、哺乳类恒温；其他动物变温。人体体温维持在37℃左右。
  • 调节机制：下丘脑为体温调节中枢。产热（骨骼肌战栗、代谢增强）和散热（皮肤血管扩张、出汗）动态平衡。

第四章 电路探秘

4.1 电荷与电流

  • 摩擦起电：用摩擦的方法使物体带电。自然界有两种电荷：正电荷（丝绸摩擦过的玻璃棒）和负电荷（毛皮摩擦过的橡胶棒）。同种电荷相斥，异种电荷相吸。
  • 电流方向：规定正电荷定向移动的方向为电流方向。在金属导体中，自由电子定向移动方向与电流方向相反。

4.2 电路的组成与连接

  • 电路基本组成：电源、用电器、开关、导线。
  • 电路状态：通路（闭合回路）、断路（开路）、短路（电源短路会损坏电源）。
  • 串联与并联：
    • 串联：各用电器依次相连，电流只有一条路径，开关控制整个电路。
    • 并联：各用电器并列连接，电流有分支，干路开关控制整个电路，支路开关控制本支路。

4.3 电流的测量

  • 电流强度（I）：单位安培（A），常用毫安（mA）、微安（μA）。
  • 电流表使用：串联在电路中，“+”进“-”出；不能直接接电源两极；读数时看清量程和分度值。
  • 串并联电路电流规律：
    • 串联电路电流处处相等：\(I = I_1 = I_2 = \cdots\)。
    • 并联电路干路电流等于各支路电流之和：\(I = I_1 + I_2 + \cdots\)。

4.4 电压的测量

  • 电压（U）：使电荷定向移动形成电流的原因，单位伏特（V）。一节干电池1.5V，家庭电路220V。
  • 电压表使用：并联在被测电路两端，“+”进“-”出；可以直接接电源两极测电源电压。
  • 串并联电路电压规律：
    • 串联电路总电压等于各部分电压之和：\(U = U_1 + U_2 + \cdots\)。
    • 并联电路各支路两端电压相等：\(U = U_1 = U_2 = \cdots\)。

4.5 电阻

  • 电阻（R）：导体对电流的阻碍作用，单位欧姆（Ω）。
  • 影响电阻大小的因素：材料、长度（越长电阻越大）、横截面积（越粗电阻越小）、温度（金属电阻随温度升高而增大）。
  • 变阻器：滑动变阻器通过改变接入电路电阻丝的长度来改变电阻。

4.6 欧姆定律

  • 欧姆定律：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。公式：\(I = \frac{U}{R}\)（适用于纯电阻电路）。
  • 伏安法测电阻：用电压表测电压，电流表测电流，\(R = \frac{U}{I}\)。

4.7 电路分析与应用

  • 串联电路：总电阻 \(R = R_1 + R_2\)，分压原理 \(\frac{U_1}{U_2} = \frac{R_1}{R_2}\)。
  • 并联电路：总电阻 \(\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}\)，分流原理 \(\frac{I_1}{I_2} = \frac{R_2}{R_1}\)。
  • 实际应用：家庭电路并联（各用电器独立），电表改装、故障分析等。
